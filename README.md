# AI Agent Lab

AI Agent와 관련된 여러 개념들을 직접 구현해보며 정리한 개인 학습 레포지토리입니다.  
LangChain을 중심으로 Agent의 기본 동작부터 상태 관리, 흐름 제어까지 단계별로 실습해 보았습니다.

---

## 📌 이 레포지토리는?

- AI Agent가 어떻게 동작하는지 직접 코드로 확인해보고
- 개념으로만 알던 기능들을 하나씩 구현해보며
- 학습 과정을 정리하기 위해 만든 개인 실습 공간입니다.

완성된 제품이나 서비스 목적이 아니라,  
**“어디까지 직접 해봤는지”를 기록하는 용도**에 가깝습니다.
